----------------------------------------------
Über pipenv zur Verwaltung von python Paketen:
----------------------------------------------


Installation von pipenv:
------------------------

1. Sicherstellen, dass pip3 installiert ist.

2. pipenv installieren: 
	pip3 install --user pipenv

3. pipenv ausführbar machen (nur falls pipenv --version nicht funktioniert):
	python user base directory bestimmen: python3 -m site --user-base
		(erwartetes Ergebnis /home/<username>/.local mit aktuellem <username>)
	Unterverzeichnis .../bin zur Umgebungsvariable PATH hinzufügen: 
		an  Datei ~/.bashrc Zeile PATH=/home/<username>/.local/bin:$PATH anhängen
	Kommando pipenv --version sollte jetzt funktionieren


Verwendung:
-----------

0. Hilfe erhalten:
	pipenv --help

1. Projekt mit python 3 anlegen: 
	Im Projektordner ausführen: pipenv --three

2. library erstmalig zum Projekt hinzufügen:
	Im Projektordner ausführen: pipenv install paketname

3. Alle fehlenden libraries des Projekts automatisch installieren:
	Im Projektordner ausführen: pipenv install

4. Befehl ausführen (in der virtualenv mit den installierten Paketen):
	Im Projektordner ausführen: pipenv run python3 beispielscript.py

5. In virtuelle Umgebung mit den Paketen wechseln um python skripte auszuführen:
	Im Projektordner ausführen: pipenv shell
